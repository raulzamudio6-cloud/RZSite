# Portafolio de Raúl Zamudio

Este es un sitio web de portafolio profesional para Raúl Zamudio, Lead Software Engineer & Technology Leader.

## Estructura
- `index.html`: Página principal
- `styles.css`: Estilos CSS
- `script.js`: JavaScript para interacciones
- `netlify.toml`: Configuración de despliegue
- `gruntfile.js`: Configuración de Grunt para build
- `package.json`: Dependencias

## Despliegue
Para desplegar en Netlify:
1. Instalar dependencias: `npm install`
2. Construir: `npm run build`
3. Subir la carpeta `dist` a Netlify

O usar la herramienta de despliegue integrada.
